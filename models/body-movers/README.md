# Body Movers
* Roblox Studio Version: 0.427.0.399257

Contains each "body mover" instance as inserted by Roblox Studio:

* BodyAngularVelocity
* BodyForce
* BodyGyro
* BodyPosition
* BodyThrust
* BodyVelocity

Some of these instances have interesting property aliases in Lua, but fairly normal serialization.
